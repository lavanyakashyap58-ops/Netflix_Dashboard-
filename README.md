# Netflix_Dashboard-
Developed an interactive Tableau dashboard to analyze Netflix movies and TV shows by genre, ratings, release year, and country distribution. Implemented dynamic filters and visualizations to identify content trends and audience insights.
# Overview

This project is an interactive Netflix Dashboard created using Tableau to analyze movies and TV shows available on Netflix. The dashboard provides insights into content distribution, ratings, genres, release trends, and country-wise availability.

# Features
Interactive filters for:
Type (Movie / TV Show)
Title selection
Country-wise visualization of Netflix content
Ratings distribution analysis
Top 10 genres visualization
Year-wise trend of Movies and TV Shows
Dynamic charts and visual storytelling
# Tools & Technologies
Tableau
Data Cleaning
Data Visualization
Dashboard Design
Analytics & Insights
Key Insights
Movies dominate Netflix content compared to TV Shows.
TV-MA and TV-14 are the most common content ratings.
Drama and Documentary genres are among the most popular categories.
Significant growth in Netflix content was observed after 2015.
Project Purpose

The goal of this project is to transform raw Netflix dataset information into meaningful visual insights that help understand content trends, audience targeting, and global distribution patterns.
